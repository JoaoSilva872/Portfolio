FaceSwap é uma aplicação web que utiliza reconhecimento facial para o entretenimento

Através de fotos de alguns voluntarios poderás colocar a imagem da cara de outra pessoa sobre a tua

Para utilizar precisas autorizar o FaceSwap a utilizar a tua camara e podes usar a tecla espaço para trocar de mascara

Prima "Espaço" para trocar a mascara
